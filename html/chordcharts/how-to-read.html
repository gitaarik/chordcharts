How to read

Measures

A box represents one measure (commonly 4 beats), which can contain one or multiple chords. The space a chord gets inside a box represents the number of beats it has to be played; a full box is 4 beats, half a box is 2 beats and quarter a box 1 beat:

Repetition

A box containing a percent sign means the previous measure should be repeated:

%

Rests

When no chords should be played, the box indicates a rest:

REST

Chords

The chords are written with their name and symbol. For a list of what which symbol means see the chord symbol list.

Alternative bass note

A chord can have an alternative bass note. Instead of playing the root note as lowest tone in the chord, you play an alternative note. In this case the chord name and symbol will be followed by a slash ( / ) and the alternative bass note. For example, this is a E7 chord with a B as the bass note:

E7/B
