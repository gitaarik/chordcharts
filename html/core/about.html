About JazzChords Our Team Contribute

JazzChords is the best place to create, share and explore chord charts for any style of music, for any instrument.

With the chart editor and search features of JazzChords, it has never been easier for groups and individuals to learn, share and remember songs.

The idea for JazzChords arose from the lack of websites that provide quality chord charts and an intuitive interface to create them.

The technology behind JazzChords is free software, which means everyone is free to read and change the source code and to contribute to make JazzChords better. If you are interested in contributing in any way, you can look on the GitHub page or contact us.
